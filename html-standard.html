Doctype

Browsers automatically assume that HTML5 in being used but the <!DOCTYPE html> at the begining of your html file will tell the browser that the latest version of HTML is being used. Any HTML file will need to be saved as an .html extension.

The <html> tag

The <html> tag tells the browser that anything inside of this tag is actual HTML elements and should be rendered as such.

The <head> tag

This tag shows the metadata of a webpage, such as the title with <title> wich will display the title of the webpage in the tab, and the link to which you link to the CSS stylesheet.

Linking to External or Internal Webpages

Linking to an external or an internal webpage uses the same concept of taking an anchor tag, <a>, and setting its href attribute to the url of the webpage if its external or its filepath if its internal. By using the <target> attribute and setting it equal to '_blank' then it will open the webpage in a new tab. It is also possible to link to a specific part on the current page by setting the href equal to the id of an element on the same page.
